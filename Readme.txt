





Bitte besuchen Sie auf jeden Fall das HTML Interface für die Applikation.
Dieses ist aufrufbar unter: ase-poi.appspot.com


Die Webservice Clients wurden als Bash Commandline Applikationen für Linux umgesetzt.
Vor Ausführung bitte nicht vergessen diesen das Linux execute Recht zu vergeben.
Auch muß das Package "cUrl" installiert sein.

Weiters können Sie auch die angefügten Videos im webm Format welche kurz die Funktionalität der Applikation in Teilen demonstrieren ansehen.


Beim ersten Mal kann der Aufruf der Applikation unter Umständen etwas lange dauern da Google App Engine den Ausführungscode erst laden muß 
wenn die Applikation mehr als eine viertel Stunde nicht aufgerufen wurde.
